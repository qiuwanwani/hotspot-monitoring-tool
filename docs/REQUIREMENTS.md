# 热点监控工具 - 需求文档

## 项目概述

**项目名称：** Hotspot Monitoring Tool（热点监控工具）

**核心目标：** 自动发现和监控热点信息，第一时间推送通知，让用户走在"吃瓜第一线"

## 用户痛点

1. 手动搜索热点信息效率低下
2. 错过重要热点信息的第一时间发布
3. 难以辨别热点信息的真伪
4. 需要同时关注多个信息源

## 功能需求

### 1. 关键词监控

**用户故事：** 作为用户，我希望能够设置关键词，当相关内容出现时第一时间收到通知。

**功能点：**
- 用户可以添加、编辑、删除监控关键词
- 支持设置关键词分类（如"AI编程"、"科技新闻"等）
- AI智能识别虚假/重复内容
- 支持设置监控频率（默认30分钟）
- 支持设置过滤条件

**验收标准：**
- [ ] 用户可以添加至少10个关键词
- [ ] 系统能在30分钟内检测到新内容
- [ ] AI能识别并过滤虚假内容

### 2. 热点发现

**用户故事：** 作为用户，我希望系统能自动发现指定范围内的热点，让我不错过重要信息。

**功能点：**
- 自动搜集指定范围内的热点
- AI评估热点热度分数
- 自动去重和聚合
- 支持历史热点回顾
- 多数据源并行获取

**验收标准：**
- [ ] 系统能同时从多个数据源获取信息
- [ ] 热点按热度分数排序
- [ ] 重复内容被自动合并

### 3. 通知推送

**用户故事：** 作为用户，我希望在热点出现时第一时间收到通知。

**功能点：**
- 浏览器推送通知
- 邮件通知
- 支持设置推送频率和时段
- 支持静默模式（免打扰时段）

**验收标准：**
- [ ] 浏览器推送能在10秒内送达
- [ ] 邮件通知能在1分钟内发送
- [ ] 静默模式下不发送通知

### 4. 数据源管理

**用户故事：** 作为用户，我希望系统能从多个可靠的数据源获取信息。

**数据源类型：**
- 网页搜索爬虫（直接爬取搜索引擎结果）
- Twitter (X) API（通过 twitterapi.io）
- RSS订阅源
- Hacker News API

**验收标准：**
- [ ] 至少支持4种数据源
- [ ] 单一数据源故障不影响整体运行
- [ ] 爬虫频率可控，避免被封禁

### 5. AI服务集成

**用户故事：** 作为用户，我希望系统能智能分析内容真伪和热度。

**功能点：**
- 支持多AI服务商接入
- 用户可自定义AI配置
- 内容真伪识别
- 热度评分

**验收标准：**
- [ ] 支持OpenRouter、OpenAI、Anthropic等主流服务商
- [ ] 支持自定义API地址
- [ ] AI分析准确率 > 80%

## 非功能需求

### 性能要求
- 页面加载时间 < 3秒
- API响应时间 < 1秒
- 支持100+关键词同时监控

### 安全要求
- API Key加密存储
- 用户数据本地存储
- 爬虫请求频率限制

### 可用性要求
- 响应式设计，支持移动端
- 暗色主题，护眼舒适
- 操作简单直观

## 数据源配置

| 数据源 | 获取方式 | 更新频率 | 备注 |
|--------|----------|----------|------|
| 网页搜索 | 爬虫 | 30分钟 | 注意频率控制 |
| Twitter (X) | twitterapi.io API | 实时 | 需要API Key |
| RSS订阅 | RSS解析 | 30分钟 | 可配置源 |
| Hacker News | 官方API | 30分钟 | 免费 |

## AI服务配置

```typescript
interface AIConfig {
  provider: 'openrouter' | 'openai' | 'anthropic' | 'custom';
  apiKey: string;
  baseUrl?: string;
  model: string;
}
```

## 通知配置

| 通知类型 | 触发条件 | 延迟要求 |
|----------|----------|----------|
| 浏览器推送 | 关键词匹配 | < 10秒 |
| 邮件通知 | 热点汇总 | < 1分钟 |

## 检查频率

- 默认：每30分钟检查一次
- 用户可自定义调整（最短5分钟，最长24小时）

## 产品形式

1. **响应式Web页面** - 主要使用形式
2. **Agent Skills技能** - 可交给其他AI使用

## 开发优先级

1. 首先完成Web版，确保功能正常使用
2. 然后开发Agent Skills技能
